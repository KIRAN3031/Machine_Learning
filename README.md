## 🎯 Overview

### Supervised Learning (SupervisedML)
Supervised learning algorithms learn from labeled training data to make predictions. This section includes:

- **Linear Regression** - Single and multiple linear regression for continuous value prediction
- **Classification** - Logistic regression and other classification techniques
- **Decision Tree** - Tree-based decision making models
- **K-Nearest Neighbors (KNN)** - Instance-based learning algorithm
- **Naive Bayes Classifier** - Probabilistic classifier
- **Random Forest** - Ensemble learning with multiple decision trees
- **Support Vector Machine (SVM)** - Powerful classification and regression algorithm
- **Regularization** - Techniques to prevent overfitting (Ridge, Lasso, Elastic Net)
- **Ensemble Learning** - Combining multiple models for better predictions

### Unsupervised Learning (unSuprvisedML)
Unsupervised learning finds patterns in unlabeled data. This section includes:

- **K-means Clustering** - Partitioning data into clusters
- **Hierarchical Clustering** - Building hierarchies of clusters
- **DBSCAN** - Density-based spatial clustering
- **Feature Extraction** - Extracting meaningful features from raw data
- **Feature Selection** - Selecting most relevant features
- **Anomaly Detection** - Identifying unusual patterns in data

## 📖 How to Use Each Section

### Supervised Learning
1. Start with **Linear_Regression** to understand basic regression
2. Move to **classification** for binary/multi-class problems
3. Explore **decision_tree** and **random_forest** for tree-based models
4. Study **Ensemble_learning** to combine multiple models
5. Check **regularization** to prevent overfitting

### Unsupervised Learning
1. Begin with **K-means-clustering** for basic clustering
2. Explore **Hierarchical-clustering** for tree-like structures
3. Learn **DBSCAN** for density-based clustering
4. Study **Feature_Selection** and **Feature_Extraction** for dimensionality reduction
5. Check **anomaly-detection** for outlier identification
